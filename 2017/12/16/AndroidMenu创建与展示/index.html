Android Menu 创建与展示

发表于 2017-12-16 | 分类于 代码 | 阅读次数: |

设置底部菜单项

@Override
public boolean onCreateOptionsMenu(Menu menu) {

    String[] m = UiUtils.getStringArray(R.array.menu);
    menu.add(Menu.NONE, Menu.FIRST + 1, 1, m[0]);
    menu.add(Menu.NONE, Menu.FIRST + 2, 2, m[1]);
    menu.add(Menu.NONE, Menu.FIRST + 3, 3, m[2]);
    menu.add(Menu.NONE, Menu.FIRST + 4, 4, m[3]);
    return super.onCreateOptionsMenu(menu);
}

add()方法的四个参数，依次是：

1、组别，如果不分组的话就写Menu.NONE

2、Id，这个很重要，Android根据这个Id来确定不同的菜单

3、顺序，那个菜单现在在前面由这个参数的大小决定

4、文本，菜单的显示文本

处理菜单事件

@Override
public boolean onOptionsItemSelected(MenuItem item) {
    switch (item.getItemId()) {
        case Menu.FIRST + 1:
            SPUtils.put(Constants.KEY_AUTO_LOGIN, false);
            startActivityFinish(LoginActivity.class);
            break;
        case Menu.FIRST + 2:
            startActivity(SetActivity.class);
            break;
        case Menu.FIRST + 3:
            startActivity(AboutActivity.class);
            break;
        case Menu.FIRST + 4:
            BaseActivity.killAll();
            break;
        default:
            break;
    }
    return super.onOptionsItemSelected(item);
}

添加虚拟按键设置

private void showNagivationMenuKey() {

    if (Build.VERSION.SDK_INT < 14) {
        return;
    }
    if (Build.VERSION.SDK_INT <= 20) {
        try {
            int flags = WindowManager.LayoutParams.class.getField("FLAG_NEEDS_MENU_KEY").getInt(null);
            getWindow().addFlags(flags);
        } catch (IllegalAccessException e) {
            e.printStackTrace();
        } catch (NoSuchFieldException e) {
            e.printStackTrace();
        }
    } else {
        try {
            Method setNeedsMenuKey = Window.class.getDeclaredMethod("setNeedsMenuKey", int.class);
            setNeedsMenuKey.setAccessible(true);
            int value = WindowManager.LayoutParams.class.getField("NEEDS_MENU_SET_TRUE").getInt(null);
            setNeedsMenuKey.invoke(getWindow(), value);
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

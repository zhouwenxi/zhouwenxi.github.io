    微软KMS激活服务器

    说明¶

    系统已经自动运行kms服务器(著名的vlmcsd开源软件),可在线激活vl版windows,office。 KMS服务器远程端口是默认的1688

    使用¶

    激活windows系统 在windows系统上,打开运行,输入cmd,以管理员身份打开windows的终端cmd窗口(或打开powershell)

    slmgr /ipk xxxx-xxxx-xxxx-xxxx-xxxx [密钥]

    slmgr /skms 192.168.x.x [NAS系统的IP地址]

    slmgr /ato

    激活ofiice软件¶

    在windows系统上,打开运行,输入cmd,以管理员身份打开windows的终端cmd窗口(或打开powershell)

    找到你的office安装目录,比如C:\Program Files (x86)\Microsoft Office\Office16 64位的就是C:\Program Files\Microsoft Office\Office16 office16是office2016,office15就是2013,office14就是2010. 然后目录对的话,该目录下面应该有个OSPP.VBS。 接下来我们就cd到这个目录下面,例如:

    cd C:\Program Files (x86)\Microsoft Office\Office15 或者cd C:\Program Files\Microsoft Office\Office15 cscript ospp.vbs /sethst:192.168.x.x cscript ospp.vbs /act

第三步设置主题

BaseAtivity.class

public class BaseAtivity extends AppCompatActivity {

    static String KEYS = "A";

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setMyTheme(KEYS);
    }


    public void setMyTheme(String keys) {

        switch (keys) {
            case "A":
                setTheme(R.style.greenTheme);
                break;
            case "B":
                setTheme(R.style.blueTheme);
                break;
            default:
                setTheme(R.style.greenTheme);
                break;
        }
    }
}

当KEYS改变时，加载子类Activity就会改变主题；正式开发应该本地保存KEYS。

MainActivity.class

public class MainActivity extends BaseAtivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);

        setContentView(R.layout.activity_main);
    }


    Boolean flag = true;

    public void click1(View view) {

        LinearLayout ll = findViewById(R.id.main_ll);
        if (flag) {
            flag = false;
            KEYS = "A";
            ll.setBackgroundColor(0xff00bf00);
        } else {
            flag = true;
            KEYS = "B";
            ll.setBackgroundColor(0xff007fff);
        }


    }

    public void click2(View view) {

        startActivity(new Intent(this, SecondActivity.class));

    }
}

在本身activity设置主题切换时，把相应控件设置相应颜色；

activity_main.xml

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:id="@+id/main_ll"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:background="?attr/actionBarBackgroundColor"
    tools:context=".MainActivity">

    <Button
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:onClick="click1"
        android:text="改变主题颜色" />


    <Button
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:onClick="click2"
        android:text="调转" />

</LinearLayout>

使用android:background=”?attr/actionBarBackgroundColor”设置字体背景颜色等

SecondActivity.class

public class SecondActivity extends BaseAtivity {
